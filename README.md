The following web application can be employed by simply creating a directory called Photos and adding photos into that directory. For the ranking algorithm to work, the naming convention should follow as written in the JSON file: img1.jpg, img2.jpg, img3.jpg, ect. Furthermore, more properties can be added to the JSON file such as Name or Age depending on which photos are bing compared. The Javascript and HTML files should be appropriatly updated if new properties are added to the JSON.
